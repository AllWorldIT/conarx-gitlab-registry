# Container Registry Agents and Flows

## Overview

The Container Registry team maintains a collection of AI-powered agents and flows to automate team-specific processes. These are hosted in a dedicated repository and integrate with GitLab Duo to provide intelligent automation for common workflows.

## Repository Location

**Repository:** [components/agents-and-flows/container-registry](https://gitlab.com/components/agents-and-flows/container-registry)

## Adding New Agents or Flows

To add new agents or flows for the Container Registry team:

1. Create the flow/agent definition in the [agents-and-flows repository](https://gitlab.com/components/agents-and-flows/container-registry)
1. Follow the existing patterns in `flows/` or `agents/` directory
1. Update any related process documentation (if needed)
